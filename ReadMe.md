# Getting Started

## Prerequisites
* Python 3.7+ (or the version you're using)
* `pip` (Python package installer)

## Installation
1. **Create a virtual environment (recommended):**

   ```bash
   python -m venv .venv
   ```
   ```bash
   .venv\Scripts\activate
   ```

2. **Install dependencies:**

   ```bash
   pip install -r requirements.txt
   ```
