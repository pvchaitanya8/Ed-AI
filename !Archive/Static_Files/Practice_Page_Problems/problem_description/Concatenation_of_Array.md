
**Problem Statement**

Given an array of integers `nums`, your task is to return the concatenation of the array with itself.

In other words, if `nums` is `[1,2,3]`, the result should be `[1,2,3,1,2,3]`.

**Example**

**Example 1:**
Input: `nums = [1,2,1]`  
Output: `[1,2,1,1,2,1]`

**Example 2:**
Input: `nums = [1,3,2,1]`  
Output: `[1,3,2,1,1,3,2,1]`

**Constraints**

- `1 <= nums.length <= 1000`
- `1 <= nums[i] <= 1000`
